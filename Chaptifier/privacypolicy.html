Privacy Policy

Chaptifier is serious about privacy. On top of the information Microsoft provides developers, Chaptifier uses Microsoft App Center to gather anonymous usage information.

Microsoft App Center is used to collect crash information, Windows version, device type and other basic feature use logs.

Chaptifier does not sell or rent your data in any way and the anonymous information is used only to help improve the app.

If you have any questions, please reach out to @mattjopete.
